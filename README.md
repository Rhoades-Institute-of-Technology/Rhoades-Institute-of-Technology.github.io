### Our Approach

We govern AI actions, not thoughts. Deterministic enforcement at the boundary — the AI thinks freely, but every action hits human-defined rules before execution.

Built by one developer with governed AI assistance. Every agent session clocked in, read the rules, and operated under constraints. 121+ sessions, zero architectural drift.
